node for rootElement not found

Symptom

shirates.core.exception.TestDriverException: node for rootElement not found.(mode=ios, source=<?xml version="1.0" encoding="UTF-8"?>

Platform

iOS

Cause

Solution

Restart iOS device or simulator.
