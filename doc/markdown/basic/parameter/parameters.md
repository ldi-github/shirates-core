In testrun files and config files you can set parameters below.

[type] **T**: tesrun file, **C**: config.json file

| category           | property                           | description                                                                             | default                                         | type | remarks                                      |

| Config             | android.configFile                 | testConfig file for android                                                             |                                                 |  T   |                                              |
| Config             | android.profile                    | profile for android                                                                     |                                                 |  T   |                                              |

| Config             | ios.profile                        | profile for iOS                                                                         |                                                 |  T   |                                              |
| Stub               | stubServerUrl                      | Stub server url                                                                         |                                                 | T/C  |                                              |
| Test mode          | noLoadRun                          | No-Load-Run(NLR) mode is enabled on true                                                | false                                           | T/C  |                                              |
| Priority           | must                               | Skip test function with `@must` on false                                                | true                                            |  T   |                                              |
| Priority           | should                             | Skip test function with `@should` on false                                              | true                                            |  T   |                                              |
| Priority           | want                               | Skip test function with `@want` on false                                                | true                                            |  T   |                                              |
| Priority           | none                               | Skip test function without priority on false                                            | true                                            |  T   |                                              |
| Log                | logLanguage                        | Language for logging                                                                    | ""(empty)                                       |  T   | Default(empty) is English. "ja" is supported |
