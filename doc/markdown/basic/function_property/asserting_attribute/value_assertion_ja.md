# value属性の検証

これらの関数を使用して要素の`value`属性を検証することができます。

### 注意

これは **iOS専用** です。

## 関数

| 関数                 |
|:-------------------|
| valueIs            |
| valueIsNot         |
| valueStartsWith    |
| valueStartsWithNot |
| valueContains      |
| valueContainsNot   |
| valueEndsWith      |
| valueEndsWithNot   |
| valueMatches       |
| valueMatchesNot    |
| valueIsEmpty       |
| valueIsNotEmpty    |
