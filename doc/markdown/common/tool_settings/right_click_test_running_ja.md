# 右クリックによるテスト実行を有効にする

### 右クリックによるテスト実行を有効にする

1. `IntelliJ IDEA > Settings` (or `File > Settings`)
1. `Build, Execution, Deployment > Build Tools > Gradle`
1. `Run tests using` を `IntelliJ IDEA` に設定する
