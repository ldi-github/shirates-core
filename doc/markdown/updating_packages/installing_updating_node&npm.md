# Installing/Updating node & npm

## Installing node

### (for Mac)

You can install **node(Node.js)** with brew. Open terminal window and type these.

```
brew install node
node -v
```

### (for Windows)

Download installation package.
https://nodejs.org/en/download/

<br>

**Note:** Use newer version to avoid troubles of installing appium.

<hr>

## Installing **n**

`n` is a version management tool for node. (https://github.com/tj/n)

```
npm install -g n
```

<hr>

## Updating n

### (for Mac)

```
n --version
npm update -g n
n --version
```

## Updating node

### (for Mac)

```
node -v
sudo n latest
node -v
```

## Updating npm

### (for Mac)

```
npm -v
npm update -g npm
npm -v
```
