# info, warn (Vision)

これらの関数を使用すると追加のメッセージを出力することができます。

| 関数   | 説明                                      |
|:-----|:----------------------------------------|
| info | **detail**のレポートでのみ出力されます                |
| warn | **simple** と **detail** の両方のレポートで出力されます |
