# デバイスの言語 (Vision/Classic)

capabilitiesを設定するとデバイスの言語を指定できます。

## 例

`testConfig.json`の `capabilities`セクションにおいて`language` と `locale`を設定します。

```
  "capabilities": {
    "language": "en",
    "locale": "US"
  },
```
