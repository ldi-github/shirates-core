# Switching AI-OCR language (Vision)

Set `visionOCRLanguage` parameter in properties file.

### English(default)

```properties
## Vision --------------------
#visionOCRLanguage=
```

Just comment it out. `en` is default.

### Japanese

```properties
## Vision --------------------
visionOCRLanguage=ja
```
