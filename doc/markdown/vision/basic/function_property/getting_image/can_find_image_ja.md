# canFindImage (Vision)

これらの関数を使用して画像テンプレートにマッチする要素を取得できるかどうかを調べることができます。関数はtrueまたはfalseを返します。

## 関数

| 関数           | 説明                                                                                  |
|:-------------|:------------------------------------------------------------------------------------|
| canFindImage | 現在の画面内で画像テンプレートにマッチする最初の要素を取得し、trueまたはfalseを返します。 `withScroll`関数内で使用するとスクロールが発生します。 |
