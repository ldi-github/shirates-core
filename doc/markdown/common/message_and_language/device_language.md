# Device language

You can specify device language setting using capability settings.

## Sample code

[Getting samples](../../getting_samples.md)

Set `language` and `locale` in capabilities section in testConfig.json

```
  "capabilities": {
    "language": "en",
    "locale": "US"
  },
```
